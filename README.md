# 🧪 Test Design – Web Application

Proyecto enfocado en el diseño de pruebas para una aplicación web, aplicando técnicas de QA para validar funcionalidades clave antes de su ejecución.

---

## 📌 Objetivo
Diseñar una estrategia de pruebas efectiva que permita identificar posibles fallos en etapas tempranas del desarrollo, asegurando la calidad del producto.

---

## 🧪 Actividades realizadas
- Análisis de requerimientos funcionales
- Diseño de casos de prueba (test cases)
- Identificación de escenarios positivos y negativos
- Aplicación de técnicas de diseño de pruebas (equivalence partitioning, boundary values)
- Documentación estructurada de pruebas

---

## 🛠️ Herramientas utilizadas
- Google Sheets / Excel (documentación)
- Jira (gestión de requerimientos)
- Navegador web

---

## 📋 Entregables
- ✔️ Casos de prueba documentados
- ✔️ Escenarios de prueba
- ✔️ Matriz de pruebas
- ✔️ Evidencia de análisis
